# mmkbr
Micro Macro Keyboard

Arduino Micro Pro powered 3x3 macro keyboard

![mmkbr](https://github.com/user-attachments/assets/6729368e-69da-4402-ad02-f016b0bce755)

# Layout

Keys are mapped to `alt` + `ctrl` + `F keys`
||||
|--|--|--|
| F13 | F14 | F15 |
| F16 | F17 | F18 |
| F19 | F20 | F21 |

# STL

Models are basic with snap on and no locking features.
